  4. Describe diffrence between using strong, em tag and b, i tag (5 point).
    Answer: b,strong은 유사하며 b는 bold의 약자로 글씨를 굵게 지정하고 strong또한 글자를 굵게 표시하여, 텍스트를 중요하게 보일때 사용한다. em과 i또한 역할이 비슷하며 em은 특정테스트를 강조하고자 할때 사용하며 i는 이탤릭체로 표시할때 쓴다.
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer: alt는 이미지를 대체하는 문구를 제공하는 속성을 갖고있으며, title은 이미지에 대한 툴팁 설명을 제공하는 속성을 갖고있다.
  6. In table, describe difference between th and td (5 point).
    Answer: th 는 header cells in table 의 의미로 표의 내용중 title 개념을 표현할때 사용된다.td 는 Standard cells in table 의미로 표의 내용중 일반적인 내용을 표현할때 사용한다.
